# Users and Groups

Job Interview assignment.


### Prerequisites

You need to have installed:

```
NodeJS
MySQL database server
```

### Installing

Usage:

1. Run commands:
```
git clone https://github.com/makz90/usersandgroups
npm install
```
3. Enter your database details in server.js
```
host: 'localhost',
user: 'testuser',
password: 'testpassword'
```
4. Start server:
```
node server.js
```
